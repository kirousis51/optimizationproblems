# An autocorrelation constant related to Sidon sets

## Description of constant

$C_1$ is the largest constant for which one has
$$
 \max_{-1/2 \leq t \leq 1/2} \int_\mathbb{R} f(t-x) f(x)\ dx \geq C_1 \left(\int_{-1/4}^{1/4} f(x)\ dx\right)^2
$$
for all non-negative $f \colon \mathbb{R} \to \mathbb{R}$.
