# Lindelof (pointwise growth) exponent for the Riemann zeta function

## Description of constant

Define the infimal exponent $\mu_\zeta$ by
$$
\mu_\zeta\ :=\ \inf\Bigl\{\theta\ge 0:\ \lvert\zeta(1/2+it)\rvert\ll_{\varepsilon}(1+\lvert t\rvert)^{\theta+\varepsilon}\ \text{for all }\varepsilon>0\Bigr\}.
$$

We define
$$
C_{62a}\ :=\ \mu_\zeta,
$$
the Lindelof (pointwise growth) exponent for $\zeta(1/2+it)$.

The Lindelof hypothesis is the conjecture that $\lvert\zeta(1/2+it)\rvert\ll t^{\varepsilon}$ for every $\varepsilon>0$, which is equivalent to $C_{62a}=0$.
<a href="#Har2019-lindelof">[Har2019-lindelof]</a>

Unconditionally, convexity gives $C_{62a}\le 1/4$.
<a href="#Har2019-convexity-1-4">[Har2019-convexity-1-4]</a>
Hardy-Littlewood proved the bound $C_{62a}\le 1/6$.
<a href="#Har2019-hl-1-6">[Har2019-hl-1-6]</a>
Bourgain proved the sharper bound $C_{62a}\le 13/84$.
<a href="#Bou2017-13-84">[Bou2017-13-84]</a>

The best established range currently is
$$
0\ \le\ C_{62a}\ \le\ \frac{13}{84}.
$$
