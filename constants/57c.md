# Univalent Bloch constant

## Description of constant

Let $\mathbb{D}=\{z\in\mathbb{C}:\lvert z\rvert<1\}$ and let $\mathcal{F}$ be the class of holomorphic functions $f:\mathbb{D}\to\mathbb{C}$ normalized by $f'(0)=1$. <a href="#BS2023-def-F">[BS2023-def-F]</a>

For $f\in\mathcal{F}$, let $B_f$ denote the radius of the largest univalent disk in $f(\mathbb{D})$. <a href="#BS2023-def-Bf">[BS2023-def-Bf]</a>

The **univalent Bloch constant** is
$$
B_u\ :=\ \inf\{B_f : f\in\mathcal{F},\ f\ \text{is univalent in }\mathbb{D}\}.
$$
<a href="#BS2023-def-Bu">[BS2023-def-Bu]</a>

We define
$$
C_{57c}\ :=\ B_u.
$$

The best rigorously stated lower bound in the cited survey literature is
$$
B_u\ >\ 0.5708858.
$$
<a href="#BS2023-latest-Bu-lb">[BS2023-latest-Bu-lb]</a>

Also, from $B\le B_l\le L\le B_u$ and $L>\frac{1}{2}+10^{-335}$, one gets the weaker bound $B_u>\frac{1}{2}+10^{-335}$. <a href="#BS2023-relations">[BS2023-relations]</a> <a href="#BS2023-bounds-L">[BS2023-bounds-L]</a>

## Known upper bounds

| Bound | Reference | Comments |
| ----- | --------- | -------- |
| $1$ | Trivial | The identity function $f(z)=z$ is univalent with $f'(0)=1$ and has $B_f=1$, hence $B_u\le 1$. |

## Known lower bounds

| Bound | Reference | Comments |
| ----- | --------- | -------- |
| $\dfrac{1}{2}+10^{-335}$ | <a href="#Yan1995">[Yan1995]</a> | Follows from $L\le B_u$ together with Yanagihara's lower bound for $L$ as summarized in <a href="#BS2023">[BS2023]</a>. <a href="#BS2023-relations">[BS2023-relations]</a> <a href="#BS2023-bounds-L">[BS2023-bounds-L]</a> |
| $0.5708858$ | <a href="#Skin2009">[Skin2009]</a> | Best recorded lower bound (as summarized in <a href="#BS2023">[BS2023]</a>). <a href="#BS2023-latest-Bu-lb">[BS2023-latest-Bu-lb]</a> |

## Additional comments and links

- **Relations to nearby constants.** The standard chain is
  $$
  B\ \le\ B_l\ \le\ L\ \le\ B_u.
  $$
